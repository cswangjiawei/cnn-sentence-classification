# Convolutional Neural Networks for Sentence Classification
This is pytorch implementation of the "Convolutional Neural Networks for Sentence Classification"

## requirement
- pytorch 1.0  
- torchtext 0.4  
- python 3.6

## results
Model|TREC|MR
--|:--:|:--:
CNN-rand|92.0|76.5
CNN-pretrain|92.6|78.7
